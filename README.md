\## ğŸ§  Theory (Brief)



Matrix multiplication is computationally expensive for large matrices.



\* In the \*\*sequential approach\*\*, all computations are done by a single process.

\* In the \*\*parallel approach\*\*, the work is divided among multiple processes, where each process computes a subset of rows.



This reduces execution time and demonstrates the practical use of OS concepts like process creation, synchronization, and workload distribution.
