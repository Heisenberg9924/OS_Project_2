## ğŸ§  ***Theory (Brief)***

Matrix multiplication is computationally expensive for large matrices.

* ***Sequential Approach:*** All computations are done using a single process.
* ***Parallel Approach:*** Work is divided among multiple processes, where each process computes a subset of rows.

This helps reduce execution time and demonstrates OS concepts like process creation, synchronization, and workload distribution.
